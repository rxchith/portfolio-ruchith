# Portfolio - Ruchith

A premium, creative portfolio built with modern web standards and high-end aesthetics.

## Features
- **Dark Mode**: Deep black background for maximum contrast.
- **Glassmorphism**: Subtle glass effects for a modern look.
- **Neon Accents**: Vibrant neon lime highlights.
- **Micro-interactions**: Smooth cursor glow and scroll animations.
- **Responsive**: Fully optimized for mobile and desktop.

## Getting Started
Simply open `index.html` in your browser.

## Built With
- HTML5
- Vanilla CSS
- Vanilla JavaScript
- Google Fonts (Inter & Outfit)
